> ## Sincronizando o git
> Na página inicial do insomnia, vá para "create" e selecione "git clone"
> ![enter image description here](https://user-images.githubusercontent.com/30088774/200962625-a2c5c991-0813-43d7-a7cb-59f78f591f85.png)
> Em seguida coloque o link do repositório dos testes
> (https://github.com/PatrickStyle66/EccApi) 
> e clique em "sync":![enter image description here](https://user-images.githubusercontent.com/30088774/200963102-7ae806e9-64ad-4b26-a308-e80792a3eb0c.png)
> ## Setando o ambiente e configurando as variáveis
> Após syncar o git é necessário escolher o ambiente, atualmente temos 2 (um para a documentação e outro para os testes unitários), na aba "debug" vá para o canto superior esquerdo onde está escrito "OpenApi env" e escolha "Dev tests ECC"
> ![enter image description here](https://user-images.githubusercontent.com/30088774/200963777-f243a4ab-cd0e-4532-94da-0f0d57109b32.png)
> em seguida, rodar os testes referentes ao cadastro. Se tudo ocorrer bem, os testes de login poderão ser realizados e as variáveis de ambiente conseguiram salvar os valores dos tokens, para verificar isso basta ir no mesmo lugar onde foi escolhido o ambiente e clicar em "Manage Environments" e em "Dev Tests ECC"![enter image description here](https://user-images.githubusercontent.com/30088774/200963791-b806309c-2937-49d6-ae90-1b67fcda10f2.png)
